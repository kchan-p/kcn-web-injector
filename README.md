# kcn-web-injector

WebページにCSS/JS/HTMLを挿入するElectronアプリ<br>
https://note.affi-sapo-sv.com/kcn-web-injector.php<br>
<br>
※開発向けのため、入力値をそのまま挿入します

### インストール

```bash
$ npm install
```

### 開発方法

```bash
$ npm run dev
```

### ビルド/実行

```bash
# For windows
$ npm run build:win
```

※ERROR: Cannot create symbolic linkエラーが表示された場合、管理者権限でビルドすること<br>
<br>
ビルド後に dist\win-unpacked\kcn-web-injector.exe を実行<br>

```bash
# For macOS
$ npm run build:mac
```

```bash
# For Linux
$ npm run build:linux
```
